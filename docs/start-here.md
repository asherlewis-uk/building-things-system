# Start Here

AI-UI-100 is the native mobile authority in a two-repository product system.

- Building.Things owns the desktop workspace shell.
- AI-UI-100 owns the native iOS and Android experience.
- A shared protocol connects the two.

Read these first:

- `docs/native-role.md`
- `docs/shared-contract-consumption.md`
- `docs/building-things-integration.md`

This repository should stay native-first and should not be reshaped into a desktop-derived shell.
